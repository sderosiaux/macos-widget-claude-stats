# macos-widget-claude-stats

A native macOS desktop widget showing your Claude Code usage at a glance. Glanceable productivity pulse.

Reads from the local claude-warehouse DuckDB database to show sessions, tokens, costs, and focus patterns.

## Planned features

- Today's session count + tokens burned
- Top 3 projects by activity
- Cost estimate (dollars)
- Focus score: deep in 2 projects vs spread across 15
- Weekly trend sparkline
- Same SwiftUI+AppKit floating window pattern as [linkedin-desktop-widget](https://github.com/sderosiaux/linkedin-desktop-widget)

## Prerequisites

- macOS 14+
- Swift 5.9+
- [claude-warehouse](https://github.com/sderosiaux/claude-plugins) DuckDB database
